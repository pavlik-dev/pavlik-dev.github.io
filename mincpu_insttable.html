LOAD

Loads the value from a memory address into the accumulator (ACC) register.

0001 <read from>

STORE

Writes arg #2 into arg #1. (#2 can be an immediate value)

0002 <target> <source>

SWAP

Swaps the values between two memory addresses.

0004 <addr #1> <addr #2>

PTR

Writes a memory address to another memory address, effectively creating a pointer.

0005 <target> <point to>

PUSH

Pushes a value onto the main stack.

0010 <value>

POP

Pops (retrieves and removes) the top value from the main stack and stores it in the accumulator (ACC).

0011

SDEPTH

Stores the current depth of the main stack in the accumulator (ACC).

0012

SCLEAR

Clears all elements from the main stack.

0013

FUNC

A flag to indicate the start of a function. It doesn't perform any operation.
You can use FUNC RETURN as

0020

RETURN

Returns from a function. Reads the return address from the function stack and jumps to that address.

0021

CALL

Calls a function located at the specified memory address. It pushes the current PC to the function stack before jumping.

0022 <func addr>

IN

Writes a value to an output device.

0030 <port> <value>

OUT

Reads a value from an input device and writes it to a memory address.

0031 <port> <target addr>

ISCONN

Checks if port is connected and sets the BOOL register accordingly.

0032 <port>

INC

Increments the value at a specified memory address by 1.

0040 <addr>

DEC

Decrements the value at a specified memory address by 1.

0041 <addr>

ADD

Writes the sum of arg #1 and arg #2 to arg #1.
Note that arg #1 can only be an address, a pointer, or a register.

0042 <summand addr> <summand>

SUB

Writes the difference between arg #1 and arg #2 to arg #1.
Note that arg #1 can only be an address, a pointer, or a register.

0043 <minuend addr> <subtrahend>

MUL

Writes the product of arg #1 and arg #2 to arg #1.
Note that arg #1 can only be an address, a pointer, or a register.

0044 <factor addr> <factor>

DIV

Writes the quotient of arg #1 and arg #2 to arg #1.
Note that arg #1 can only be an address, a pointer, or a register.

0045 <dividend> <divisor>

SHL

Shifts the bits of arg #1 to the left by arg #2. The vacated bits are filled with zeros.
Note that arg #1 can only be an address, a pointer, or a register.

0044 <arg #1> <arg #2>

SHR

Shifts the bits of arg #1 to the right by arg #2. The vacated bits are filled with zeros.
Note that arg #1 can only be an address, a pointer, or a register.

0044 <arg #1> <arg #2>

NEG

Negates the value of arg #1.
Note that arg #1 can only be an address, a pointer, or a register.

0044 <arg #1>

AND

Performs a bitwise AND operation between arg #1 and arg #2.
Note that arg #1 can only be an address, a pointer, or a register.

0045 <arg #1> <arg #2>

OR

Performs a bitwise OR operation between arg #1 and arg #2.
Note that arg #1 can only be an address, a pointer, or a register.

0046 <arg #1> <arg #2>

XOR

Performs a bitwise exclusive OR operation between arg #1 and arg #2.
Note that arg #1 can only be an address, a pointer, or a register.

0047 <arg #1> <arg #2>

EQ

Compares two values for equality. It sets the BOOL register to 1 if they are equal, and 0 if they are not.

0060 <arg #1> <arg #2>

NE

Compares two values for inequality. It sets the BOOL register to 1 if they are not equal, and 0 if they are equal.

0061 <arg #1> <arg #2>

GT

Compares two values to check if the first value is greater than the second. It sets the BOOL register accordingly.

0062 <arg #1> <arg #2>

LT

Compares two values to check if the first value is less than the second. It sets the BOOL register accordingly.

0063 <arg #1> <arg #2>

GE

Compares two values to check if the first value is greater than or equal to the second. It sets the BOOL register accordingly.

0064 <arg #1> <arg #2>

LE

Compares two values to check if the first value is less than or equal to the second. It sets the BOOL register accordingly.

0065 <arg #1> <arg #2>

JMP

Unconditionally jumps to a specified memory address, changing the program counter (PC).

0070 <addr>

JC

Jumps to a specified memory address if the BOOL register is set to 1 (true).

0071 <addr>

HLT

Halts the CPU execution.

00FF
